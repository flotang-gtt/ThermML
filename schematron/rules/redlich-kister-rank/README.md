# Redlich-Kister Rank

Rule family:

- `VAL-ERR-RK-RANK-INVALID`
- `VAL-WARN-RK-HIGH-RANK`
- `VAL-WARN-RK-RANK-GAPS`

Files in this directory keep the assert and warning reports with their fixtures.

- `rule.sch`: Schematron rule family
- `invalid.xml`: failing fixture for the assert
- `warning.xml`: passing fixture that emits warnings
