Validation Rules

This page is the entry point for the published validation rule surfaces. Detailed rule inventories live in the linked catalogs below.

Published Rule Surfaces

  • XSD Validation Catalog — XSD-enforced rules worth naming as public constraints.
  • Schematron Validation Catalog — Schematron rule families kept with runnable fixtures.
  • schematron/README.md — runnable Schematron layout, fixture conventions, and combined CLI integration.

Catalog Policy

  • XSD rules that are intentionally hoisted from reusable types, local declarations, or key/keyref logic belong in the published catalog.
  • Schematron rules that are intended to be part of the published validation surface belong in the published catalog and must have discoverable source files and sample fixtures.
  • The repository-shipped CLI validation flow is the combination of XSD plus the bundled Schematron entrypoint.
